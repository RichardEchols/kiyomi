Kiyomi Bot v4.0

INSTALLATION INSTRUCTIONS:
===========================

MacOS (Recommended):
1. Double-click "Install Kiyomi.command"
2. Follow the setup wizard in your browser
3. Connect your AI subscription (Claude, Gemini, or ChatGPT)
4. Create a Telegram bot via @BotFather and paste the token

Manual Installation:
1. Run: ./install.sh
2. Follow the setup wizard at http://127.0.0.1:8765

WHAT YOU NEED:
- macOS 12+ with Python 3.10+
- A Telegram account
- An AI subscription (Claude Max, ChatGPT Plus, or Gemini Advanced)
  OR a free Gemini API key from https://aistudio.google.com
