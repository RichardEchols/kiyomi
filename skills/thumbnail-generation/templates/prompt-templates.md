# Thumbnail Prompt Templates

Ready-to-use prompts for AI image generation. Customize the bracketed sections.

## Tutorial/Educational

### Basic Tutorial
```
Professional content creator at modern desk with [computer/equipment], 
demonstrating [skill], clean minimalist background, bright studio lighting, 
educational YouTube style, friendly approachable mood, 16:9 composition, 
space on right side for text overlay
```

### Coding/Tech Tutorial
```
Close-up of [laptop screen/hands on keyboard] with colorful code visible, 
dark background with subtle tech elements, blue and purple accent lighting, 
modern developer aesthetic, YouTube tutorial thumbnail style, 16:9 format, 
dramatic lighting on screen
```

### DIY/Craft Tutorial
```
Hands working on [craft project], top-down flat lay composition, 
bright even lighting, craft supplies arranged aesthetically around the frame, 
Pinterest-worthy style, warm inviting colors, 16:9 horizontal format
```

## Universal Modifiers

Add these to any prompt for YouTube-optimized results:

```
YouTube thumbnail style, bold punchy colors, 
attention-grabbing composition, high contrast, 
viral video aesthetic
```

For text space:
```
space on [left/right/bottom] for text overlay
```

For faces:
```
exaggerated facial expression, clear visible eyes, 
genuine emotion
```
